# Code TileMill is built from open source projects with the following projects playing a key role: - [Mapnik](http://www.mapnik.org) Map rendering. - [Backbone](http://github.com/documentcloud/backbone) MVC framework for structured AJAX applications. - [Express](http://github.com/visionmedia/express) Sinatra-inspired node.js web server. - [mess.js](http://github.com/developmentseed/mess.js) MSS-to-Mapnik XML map style compiler. - [tilelive.js](http://github.com/developmentseed/tilelive.js) High-speed tile-rendering server.
